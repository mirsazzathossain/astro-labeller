Welcome to Radio Galaxy Classifier's (RGC) Data Labeller Tool

RGC is a Radio Galaxy Classifier named after Radha Gobinda Chandra (1878–1975), a Bangladeshi-Indian variable star observer from the first half of the twentieth century.

RGC uses convolutional neural networks to classify radio galaxies (galaxies that emit radiation in radio wavelengths). It incorporates the current deep learning models available in astronomical literature.

This tool is a web application that allows users to label radio galaxies in different categories. In our case, we are using the following categories:

  • Narrow Angle Tailed (NAT): The angle between the lobes of galaxies is less than 90◦
  • Wide Angle Tailed (WAT): The angle between the lobes of galaxies is more than 90◦
If the galaxy is not in any of the above categories, it will be labelled as "Trash".
